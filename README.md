Структуры и объединения. Сортировка массива структур. Работа с файлами.

Вариант 11.

Определить комбинированный (структурный) тип для представления анкеты ребенка, состоящей из его имени, пола, роста, даты рождения. Ввести информацию по 20 детям. Вывести средний рост 12-летних девочек. Перенести в отдельный массив 3 самых младших мальчиков. Отсортировать список учеников по дате рождения. Вывести все данные по конкретному ученику (по фамилии ученика). Реализовать функцию изменения данных конкретного ученика. В отдельный массив поместить список детей, имеющих день рождения заданном месяце. Реализовать вывод отфильтрованных данных в виде оберточной функции.

Требования к решению:

Использовать функции для каждой подзадачи.
1. Как минимум одно поле должно иметь перечисляемый тип данных (enum).
2. При описании структуры использовать в качестве одного из полей структуру.
3. Ввод данных осуществлять в виде инициализации массива структур внутри программы (не с клавиатуры).
4. Алгоритм сортировки — пузырьковая сортировка.
5. Использовать библиотеку fstream.
6. Оформить код чтения/записи в файл с использованием подпрограмм.
